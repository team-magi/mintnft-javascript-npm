# mint-nft npm package

## IPFS

Use 3rd party Pinata platform to manage IPFS, please register [Pinata](https://pinata.cloud) first. 

## Installation

Using npm:

```bash
$ npm install mint-nft
```

Using yarn:

```bash
$ yarn add mint-nft
```
